Privacy Policy

Effective date: June 5, 2026

This Privacy Policy applies to GPS Location Changer, an Android application designed for mock GPS and route testing. The app allows users to select coordinates, save test locations, and run mock location tests through Android's official mock location provider workflow.

Information We Collect or Use

GPS Location Changer does not require users to create an account. The app may use or store the following information to provide its features:

  • Current device location, when permission is granted, to center the map or reset to the user's current area.
  • Selected latitude and longitude values entered manually, selected on the map, or chosen from search results.
  • Saved locations, including names, folders or tags, favorite status, and coordinates.
  • Route simulation data, such as route stops, movement profile, speed settings, and route progress.
  • Service status logs used for troubleshooting mock location test sessions.
  • App settings such as language, theme, units, and coordinate format.

How Information Is Stored

Saved locations, app settings, route test data, and service logs are stored locally on the user's device. GPS Location Changer does not operate a cloud account system for this information in the current version.

If the user uninstalls the app or clears app data through Android settings, locally stored information may be deleted.

Location Permission

GPS Location Changer requests location permission so it can show the user's current area on the map and support location testing workflows. The app also uses location access when running a foreground mock location test session.

The app does not request background location permission in the current version.

Mock Location Testing

GPS Location Changer uses Android's official mock location provider workflow. The user must manually select GPS Location Changer as the mock location app in Android Developer Options before mock location testing can run.

The app does not secretly override location services. Mock location testing is started and stopped by the user.

Foreground Service and Notifications

When mock location testing is active, the app may run a foreground service and show a persistent notification. This is used to keep the active test session visible and to provide a way to stop the session.

Google Maps and Places

GPS Location Changer uses Google Maps and Google Places features to display maps, search for addresses or places, and show location suggestions. These services are provided by Google and may process information according to Google's privacy policy.

Google's privacy policy is available at https://policies.google.com/privacy.

Exported Reports

The app can generate a test session report when the user chooses to export one. A report may include selected coordinates, mock location status, app settings, and service log entries.

If the user shares or uploads an exported report using another app or service, that external app or service may handle the report according to its own privacy policy.

Data Sharing

GPS Location Changer does not sell personal information. The app does not share saved locations, route test data, or service logs with an app-operated server in the current version.

Some information may be processed by third-party services, such as Google Maps and Google Places, when the user uses map and search features.

Children's Privacy

GPS Location Changer is intended as a developer, QA, educational, and technical testing utility. It is not directed to children under 13, and the app does not knowingly collect personal information from children.

User Choices

Users can control their information and app behavior in the following ways:

  • Deny or revoke location permission through Android settings.
  • Stop mock location testing at any time in the app or through the notification controls.
  • Delete saved locations and clear service logs inside the app.
  • Clear all app data through Android system settings.
  • Remove GPS Location Changer as the selected mock location app in Developer Options.

Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. If changes are made, the effective date at the top of this page will be updated.
